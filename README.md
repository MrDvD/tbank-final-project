# Итоговый проект — T-News

Этот репозиторий содержит исходный код и инструкции по запуску приложения T-News, разработанного в рамках экзаменационного задания.

## Описание приложения

- **Главная страница (Лента):** отображает посты всех пользователей, на которых оформлена подписка. Для каждого поста доступны лайки и комментарии. Комментарии представлены в виде плоского списка; только автор может удалить свой комментарий.
- **Профиль пользователя:** содержит информацию о пользователе (аватар, имя, биография), кнопку подписки/отписки, поле для создания нового поста (доступно только владельцу профиля), а также список его постов. Удалять посты может только автор.
- **Страница входа/регистрации.**
- **Страница поиска:** позволяет искать пользователей и посты (с возможностью выбора критерия поиска).
- **Настройки аккаунта:** позволяют изменять профиль и настройки пользователя (аватар, имя, биография).

## Используемые технологии

- **Vite** — инструмент сборки и разработки веб-приложения.
- **TypeScript** — основной язык программирования для фронтенда и бэкенда.
- **Fastify** — фреймворк для создания API на Node.js.
- **SCSS** — препроцессор CSS, используется для работы с переменными и медиа-запросами.
- **Docker** — контейнеризация приложения и зависимостей.
- **Docker Compose** — для управления инфраструктурой и контейнерами.
- **PostgreSQL** — реляционная база данных.
- **postgres.js** — библиотека для взаимодействия с PostgreSQL из Node.js.

## Предварительная настройка

Перед запуском приложения необходимо задать следующие переменные окружения:

```env
POSTGRES_HOST=tbank-final-project-database-1 # имя активного контейнера с базой данных
POSTGRES_USER=postgres                       # имя пользователя PostgreSQL
POSTGRES_PASSWORD=12345                      # пароль пользователя PostgreSQL
APP_SECRET=67890                             # секретный ключ для приложения
APP_PORT=3000                                # порт, на котором будет работать приложение
```

## Инструкция по запуску

1. Клонируйте репозиторий:
  ```bash
  git clone <URL-репозитория>
  ```
2. Перейдите в ветку `dev`:
  ```bash
  git checkout dev
  ```
3. Соберите контейнеры:
  ```bash
  docker compose build
  ```
4. Запустите приложение:
  ```bash
  docker compose up
  ```
5. Откройте в браузере `http://localhost:<APP_PORT>/` для тестирования приложения.
